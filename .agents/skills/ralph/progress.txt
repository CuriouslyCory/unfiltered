## Codebase Patterns
- Header component is at src/app/_components/theme/header.tsx (client component with mobile menu state)
- Layout metadata is in src/app/layout.tsx using Next.js Metadata export
- Uses shadcn/ui components from ~/components/ui/ and app-level components from src/app/_components/
- Tailwind theme tokens: text-muted-foreground, border-border, bg-muted, bg-accent for consistent theming
- The project uses a `~/` path alias that maps to `src/`
- Dev server: `pnpm dev` on port 3000
- Typecheck: `npx tsc --noEmit`
- Document types are defined in Prisma schema as enum: EXECUTIVE_ORDER, FACT_SHEET, REMARKS, LEGISLATION, OTHER
- Human-readable type labels are in src/app/_components/document-type-badge.tsx (typeConfig map)
- shadcn UI components installed via: `pnpm dlx shadcn@latest add <component>`
- Document detail page: src/app/eo-summary/[slug]/page.tsx (server component)
- ArtifactSectionList manages collective open/close state, ArtifactSection accepts optional onToggle prop for controlled mode
- URL param `?sections=all` expands all sections; individual section names comma-separated in `sections` param
- Button component: `~/app/_components/ui/button` with variant="outline" size="sm"
- document-type-badge.tsx bg-gray-500 for OTHER type is intentional (distinct badge color, not theme gray) — do not replace
- TOC is integrated inside ArtifactSectionList (which manages open/close state), not as a standalone sibling component
- Artifact section elements have id attributes derived from title: `title.toLowerCase().replace(/\s+/g, "-")` — used for scroll targeting and IntersectionObserver
- Grid layout for TOC + artifact list: `md:grid md:grid-cols-[200px_1fr] md:gap-6` on the ArtifactSectionList wrapper
- Risk border colors: border-l-green-500 (0-2), border-l-yellow-400 (3-4), border-l-orange-500 (5-6), border-l-red-500 (7-8), border-l-red-600 (9-10)
- DocumentTypeBadge defaults to `w-full` — use `className="w-auto"` when inline
- DocumentType enum (runtime value for z.nativeEnum): import from `~/generated/prisma/client`; type-only: `import { type DocumentType } from "~/generated/prisma/client"`
- RiskScore component supports `size` prop: "compact" (default, h-8 w-8 circle) and "full" (large number + label + meter bar)
- TooltipProvider is wrapped at the layout level (src/app/layout.tsx) inside TRPCReactProvider
- RiskScore is a client component ("use client") due to Tooltip dependency
- Heading font CSS variable is `--font-heading` (not --font-cactus), referenced in globals.css as `--font-serif` fallback chain
- Semantic artifact color tokens available: --color-artifact-info, --color-artifact-warning, --color-artifact-danger, --color-artifact-action, --color-artifact-share
